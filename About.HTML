How to Use:

  1. Enter the name of the city you want to check the weather for in the search bar.
  2. Press the "Search" button or hit Enter.
  3. The main section will update to display detailed weather information for the specified city.
  4. Scroll down to view weather information for other popular locations.

Key Features:

  • City Search: Users can input the name of a city they want to check the weather for using the search bar in the navigation.
  • Weather Display: Upon submitting a city name, the app dynamically updates the main section to display detailed weather information for the specified city, including temperature, humidity, wind speed, cloud cover, sunrise, and sunset times.
  • Other Places: The app also provides weather data for several other popular cities, allowing users to compare weather conditions across different locations.
  • Responsive Design: The app is built using Bootstrap, ensuring a responsive design that adapts seamlessly to various screen sizes and devices, providing a consistent user experience.

Description:

The Weather App is a user-friendly tool that allows users to access current weather conditions, including temperature, humidity, wind speed, cloud cover, and more, for any city they choose. With an intuitive interface and clean design, users can quickly navigate through the app to find the weather information they need.

With its simplicity and efficiency, the Weather App is your go-to destination for all your weather-related queries, whether you're planning a trip, checking the weather for outdoor activities, or simply staying informed about the current weather conditions.
